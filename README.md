# Elevate Webworks 🌐

Welcome to the repository for **Elevate Webworks**—a premium, custom web design, development, and SEO agency dedicated to helping growing businesses in India establish a powerful online presence.

This project is the official portfolio and business site for **Elevate Webworks**, founded by **Muthassir**.

---

## ✨ Features

- **Responsive Design**: Flawless visual experience across all screen sizes—mobile, tablet, and desktop.
- **Dark/Light Mode**: Smooth, dynamic theme toggle with system preferences memory.
- **Modern Typography & Icons**: Uses Google Fonts (Poppins) and Font Awesome icon library for a clean, professional look.
- **Service Showcases**: Clear details on web development, UI/UX design, SEO optimization, e-commerce, and maintenance.
- **Transparent Pricing**: Detailed starter, growth, and pro packages suited for different business scales.
- **Direct Lead Capture**: Integrated with custom Google Forms and quick-action WhatsApp/Email contact links.

---

## 🛠️ Built With

* **HTML5**: Semantic markup for high accessibility and SEO friendliness.
* **CSS3**: Premium custom stylesheets (Vanilla CSS) featuring modern grid layouts, flexbox, and smooth transition animations.
* **JavaScript (ES6)**: Lightweight, fast scripting for interactive elements (theme switching, typing text animations, and mobile menu toggle).

## 🚀 Getting Started

Since this is a static website, running it locally is incredibly simple:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/elevateweb-design/Elevate-Webworks.git
   ```

2. **Open the project:**
   Simply double-click the `index.html` file or use a local development server (such as Live Server in VS Code) to open the project in your browser.
